# 03 — 20 Higgsfield Video Prompts

Each prompt is **copy-paste ready** for Higgsfield. Prompts follow Higgsfield's best-practice structure: `[Scene] + [Subject] + [Action] + [Camera movement] + [Lighting] + [Style] + [Mood] + [Duration]`.

**Default settings for all:**
- Aspect ratio: **9:16 (vertical)** for Reels/Shorts
- Duration: **5–8s per clip** (stitch 2–3 clips in CapCut for longer Reels)
- Style preset: **"Cinematic"** or **"Documentary"**
- Motion: **"Smooth"** / **"Slow-mo"** where noted

## Tips for getting best results from Higgsfield

1. **Run each prompt 2–3 times** — AI video is stochastic. Pick the best take.
2. **Always regenerate if you see**: distorted hands (common), misspelled text on labels (very common), unnatural spice physics.
3. **For text on screen** (e.g. jar labels), add it in CapCut later — AI text is unreliable.
4. **For voiceovers**, use ElevenLabs (Hindi female voice "Monika" or English "Sarah"). Script included in each video concept.
5. **Add music in CapCut**: search "Indian traditional instrumental", "sarangi", or trending Reel audios (check weekly).
6. **Color-grade consistently** — use the same LUT across all videos for brand cohesion. Recommended: *"Warm Film"* preset.
