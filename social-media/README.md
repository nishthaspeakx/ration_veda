# Ration Veda — Social Media Playbook

A complete, ready-to-execute social media system for Ration Veda.

## Goals
1. **Brand awareness** — position Ration Veda as India's most authentic hand-pounded spice brand
2. **Direct WhatsApp orders** — drive buyers to +91 97117 84375

## Platforms
Instagram • YouTube Shorts • Facebook • LinkedIn • X (Twitter)

## Posting Cadence
- **Daily** — 1 post per day across all platforms (30/month)
- **20 videos/month** (Higgsfield-generated Reels/Shorts)
- **10 static/carousel posts/month**

## Files in this folder

| File | What it is |
|------|-----------|
| `01-strategy.md` | Positioning, voice, content pillars, platform strategy |
| `02-content-calendar.md` | 30-day daily calendar with post type, theme, CTA |
| `03-higgsfield-video-prompts.md` | 20 copy-paste ready Higgsfield prompts |
| `04-captions-library.md` | Platform-specific captions + hashtags for every post |
| `05-reel-scripts.md` | Shot-by-shot scripts for hero videos (15s / 30s / 60s) |
| `06-higgsfield-guide.md` | Which Higgsfield plan to buy + how to use it |

## How to use this system (weekly workflow)

**Sunday (30 min) — Plan the week:**
1. Open `02-content-calendar.md`, look at the 7 upcoming days
2. Note which 4–5 days are video days (rest are static)

**Monday (2 hrs) — Generate videos:**
3. Open `03-higgsfield-video-prompts.md`
4. Copy the week's 4–5 video prompts into Higgsfield
5. Generate, download, save to a folder named by week (e.g., `week-3/`)

**Tuesday (1 hr) — Create static posts:**
6. Shoot/select static images from `website-assets/images/`
7. Use Canva templates (links in strategy doc)

**Daily (10 min) — Schedule & post:**
8. Copy caption from `04-captions-library.md`
9. Adapt for platform (shorter for X, professional for LinkedIn)
10. Schedule via Meta Business Suite (IG+FB) + native YT/LinkedIn/X
